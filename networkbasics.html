Network types

Local area Network (LAN) : devices that are connected within ig a local network work.

Wide Area Network (WAN) : this is a network that that is ig just bigger it can connect various other LAN'S into the same WAN. an example of this would be the internet it is basically a giant complier of various local networks kinda.

Wireless local area Network (WLAN) : like lan but for device that are connected wirelessly

Storage area Network (SAN) : a network for storage devices.

Virtual private network (VPN) : secure point to point connect between 2 network endpoints ; can allow you to connect to a group of servers that are firewalled off, or just remotely connect to a local network. it can do alot more too.

Some definitions

node : a network type that can send, receive, store, and create data.

router : can be physical or virtual, a router sends information (data packets) between networks. (in my experience it can be a hassle to set up a virtual router even though doing so allows you much more control over the system going physical its always easier at the end of the day.)

switch : device that in a way "clusters" together other device allowing them to communicate more efficiently. their are 2 main kinda a managed switch and unmanaged, managed switches normally support things like qos, vlan, IGMP and Link aggregation. where as unmanaged switches are basically just like a addon that allows you to add more ethernet ports they really don't do anything other then that.

ports : its best to think of then as like physical ports you would find on a router, they allow connections weather that be incoming or outgoing to plugin to that port and use it to transfer information their are a ridiculous amount of ports, somewhere around 65,530 something i think idk.

Firewall : network security device that monitors incoming and outgoing connects to a network ensuring that unauthorized users don't have access.

load balancers : distribute network traffic across devices, and do it efficiently.
