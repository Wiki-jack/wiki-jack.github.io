Linux File System Directories

/bin

the bin directory contains binaries, basically it contains some of the apps and programs you can run.

you can find the "ls" program (lists file directories) in the /bin dir.

/boot

the /boot dir contains files requried for system boot.

/dev

/dev contains device files, generated at boot. EX. if you plugged in a usb device a new enrty will be added in the /dev dir.

/etc

/etc is a dir, you could say it stands for "Everything configurable", it contains most of the system wide configuration files.

/home

contains users personal directories.

/lib

/lib is also called "libraries", contain files of code your apps can use. EX. contains snippets of code apps can use to draw windows on your desktop.

Their are more lib dir around the system, one special lib file is "/is" it contains all important kernel modules, this includes drivers for video cards, sound cards, wiki, printer, etc.

/media

/media is a dir where external storage will be auto mounted. EX. you plug in a usb that has a fat32 partition format, that partition containing your files will most likey be found in this dir.

/mnt

/mnt file dir is where manually mounted devices or partions are put, its not ofen used anymore.

/opt

/opt file dir is often where software you compile sometimes is placed by the system. Some files may also land in /usr/local dir, where files land is completely determined by the develeper of the application.

/prof

/proc is a virtual dir, it contains info about your computer. Ex. info about your cpu and linux kernel your system is running.

/root

/root is the home dir of the superuser(aka administrator), it is sepreate from user home directories.

/run

/run is a dir that is used by system processes and is used to store temp data for various reasons.

/sbin

/sbin is simular to /bin however it contains apps that only the superuser will need, these apps can be used with the "sudo" command or by accessing the super user account and running files that way.

/usr

/usr dir is where the users home dir were orignally kept on the unix system. now replaced by the /home dir, now you can find various stuff in the /usr dir including apps, libarie, documents, wallpapers, icons, and various other things that need to be shared by apps and services.

You will also find bin, sbin, and lib dir in the /usr dir.

/srv

/srv dir is another virual dir, also contains info from devices connected to your computer. In some cases those devices can be manipulted in this dir.

/tmp

/tmp contains temp files, normally things are placed in this dir by running applications so they can be used tempularly.

/var

/var was orignally used for varibles, meaning that files in it changed frequently, now many file dir do the same thing as the /var dir so not all varible files can be found in it.

This dir normally contains system logs, and spools "tasks" that are sent to a shared printer that is also doing other jobs.
