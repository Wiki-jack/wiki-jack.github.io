Navigation

cd : allows you to move into dir

ls : list content of dir

pwd : prints path to dir

mv : move specfied file or folder/dir

find : finds stuff

mkdir : make dir/folder

rmdir : deletes dir

/etc : location most system files are stored

/var : one of the main root folders

/root : folder for "root" user

/tmp : dir used to store temporary files

File Management

touch (name) : creates text file with specified name

cp ( file you want to copy ) ( where you want to copy it to ) : copyies file to specified place

mv : move or renames specfied dir/folder or file

rm -r : deletes folder or file and all its contents

wget : allows you to download some things based on their web url

lsblk : shows drives and their directories

cat : shows content in files

scp : transfers files securly through ssh

Package manager

apt : normally default package manager on debian based systems

apt commands

apt install (package name) : installs package

apt remove (name) : remove package

apt update -y : updates with yes to all

apt upgrade -y : updates and says yes to all

apt search : searches for packages

snap : can be added to pretty much any distro with a little work but is default to ubuntu systems except turnkey containers and maybe docker

snap commands

snap install : installs specfied package

snap remove : removes specified packages

flatpak : can be pretty much added to any distro

flatpak commands

flatpak install : installs package

flatpak remove : removes packages

dnf : normally default on fedora and RHEL systems

dnf commands

dnf install : installs package

dnf remove : removes package

pacman : normally default on arch based systems

pacman commands

pacman -S : installs specified package

pacman -Rsc : removes specfied package

pacman -Ss : seaches for specified package

pacman -Syu : updates pretty much everything

zypper : Normally found on opensuse based systems

zypper commands

zypper patch : patches system

zypper install : installs package

zypper search -t : searches for package

zypper remove : removes packages

Sys Info

$ : used to indicate a command is being ran as admin

" ~ " : used to represnt users home dir

uname -a : lists info about system

neofetch : lists sys info ( installed program )

htop : displays info about sys info and resources ( installed program )

uptime : shows how long the system has been up for

lsblk : lists drives and partition

mc : cli based file manager ( midnight commander )

ps : provides list of running system process

top : real time stats of sys programs

kill : kills process ( EX. kill pid# )

systemctl : system management and service management ( EX. sudo systemctl status apache2 )

systemctl commands :

  • systemctl subcommand argument

  • restart : self explainatory

  • reboot : reboots

  • poweroff : shuts down

  • set-default : sets a service as default

  • rescue : switchs to rescue mode for troubleshooting

  • start : starts service

  • stop : stops service

  • mask : prevents service from starting for any reason

  • enable : allows me to start service at boot

  • disble : prevents me from starting service at boot

  • is-active : prints current and start up status of service

  • is-enable : same but for enabled command

Network Management

ifconfig : shows network stats ig

ip address : shows network

whois ( domain ) : shows info about specfied domain

ping ( ip ) : allows you to ping ip's and domains

netstat : display network info

traceroute ( ip ) : display route that packages take

ssh: ssh (user)@(ip address)

Users

whoami : just shows what user you are logging in as

su : allows you to change user

sudo : allows me to use the sudo commands aka do things as root user ( only able to use sudo if user is in the sudo group )

chmod : changes permissions of file or dir ( EX. chmod +x (file name) allows you to excute bash scripts however file must be specified with .sh at end and maybe a shebang < #!/bin/bash> to be excuted as such)

Misc

clear : just clears the terminal
