Programs

ping : used to test the connection between computers or networks.

traceroute (destination) : used to find the path that a request takes as it heads to another machine.

whois : allows you to find out who a domain is registered to EX. whois google.com

DIG : allows me to manually see recursive DNS servers of out choice for info about domains EX. dig (domain)@(DNS-server-ip)

Nmap : network scanner

Wireshark : network protocol scanner

Angry IP Scanner : ip address manager and port scanner

ufw : uncomplicated firewall (linux program)

OPNsense : Firewall/Router

pfsense : firewall/router

ipfire : firewall/router (I think thats what it is called cant remember dont use often)

Snort : Intrusion Prevention System

Zibbit : network monitor ( i think thats what its called never ran it coudnt get it to work)

Pihole : network wide ad blocker

PiAlert : manages who connects to my local network

Observium : network monitor this is the one i use its great
